<objective>
Complete the UUID authentication flow by wiring school UUID into JWT tokens and adding UUID-based user lookup to the auth repository.

**Purpose:** JWT tokens currently contain the `user_uuid` claim but not `school_uuid`. This plan populates school UUID in tokens so downstream middleware can use UUID-based tenant resolution. It also adds `FindByUUID` to the auth repository for UUID-based user lookup (needed by Phase 3+ API compatibility).

**Output:**
- AuthRepository.FindByUUID method
- TokenService passes schoolUUID to JWT service
- AuthService extracts school UUID from user's school associations during login
- All token generation paths carry school UUID
</objective>
