    1. **Lock check**: Acquire `mu.Lock()`, check if `inProgress[schoolID]` is true (return ErrBackupInProgress). Set `inProgress[schoolID]=true`. Release lock via defer.

    2. **Get school connection**: Load `models.SchoolConnection` from coreDB where ID=schoolID. If not found, return ErrSchoolNotFound.

    3. **Build DSN** for pg_dump using school connection fields directly (NOT via ConnectionManager which creates a GORM connection — we need the raw DSN string for pg_dump):
       ```
       host={school.DatabaseHost} port={school.DatabasePort} user={school.DatabaseUsername} password={decryptedPassword} dbname={school.DatabaseName} sslmode=disable
       ```
       Decrypt `school.DatabasePasswordEncrypted` using the same pattern as connection_manager.go (use crypto.Service). The executor needs access to the crypto service. **Pass `*crypto.Service` as a dependency to BackupService.**

    4. **Update status to 'running'**: Update tenant_backups record or set in-progress marker. The record doesn't exist yet — create it with status `"running"`.

       - Build command: `exec.Command("pg_dump", "-Fc", "--dbname="+dsn, "-f", tempFile.Name())`

       - Run cmd with timeout (30 minutes via context.WithTimeout)
       - If err, mark backup as 'failed', return ErrPGDumpFailed + stderr output

    6. **Upload to S3**:
       - Timestamp: `time.Now().UTC().Format(time.RFC3339)`
       - Key: `backups/{schoolID}/{timestamp}/{dumpName}.dump`
       - Open temp file: `os.Open(tempFile.Name())`
       - Get file stats for size
       - Call `s3Storage.Save(key, file)`
       - Close and remove temp file

    7. **Record metadata**: Create `models.TenantBackup` record via coreDB:

       ```go
       backup := models.TenantBackup{

           RestorePoint: &now,
       }
       ```
       Save via `cm.coreDB.Create(&backup)`.

    8. **Enforce retention**: Count completed backups for this school. If > 14, find the oldest backups beyond the 14th, delete from S3, then delete from DB. Per D-07.

    9. **Return result.**

    **Helper: `EnforceRetention(ctx context.Context, schoolID uint) error`** — counts completed backups, deletes oldest beyond 14.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| BackupService → S3 | Backup data sent to external S3 bucket |
| pg_dump execution | Tenant DB credentials used in subprocess |
| API → BackupService | Admin API triggers manual backup |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Information Disclosure | S3BackupStorage | mitigate | Load AWS credentials from env vars only (S3_ACCESS_KEY, S3_SECRET_KEY); never log S3 credentials |
| T-05-02 | Information Disclosure | BackupService | mitigate | Sanitize DSN before logging: log only `dbname={dbname} host={host}` without password |
| T-05-03 | Elevation of Privilege | CreateTenantBackup | mitigate | Enforce school_id scoping — school A cannot create backup for school B |
| T-05-04 | Denial of Service | pg_dump execution | mitigate | Set context timeout of 30 minutes; concurrent backup per school is prevented via inProgress map |
| T-05-05 | Tampering | S3 backup storage | accept | S3 bucket-level security (IAM policies, versioning) is infrastructure concern outside backup service scope |
</threat_model>
