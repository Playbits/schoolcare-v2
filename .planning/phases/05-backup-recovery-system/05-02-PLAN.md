       **`ListBackups(ctx context.Context, schoolID uint) ([]models.TenantBackup, error)`:**
       Lists all completed backups for a school, ordered by created_at DESC.

       **`RestoreTenantBackup(ctx context.Context, schoolID, backupID uint) error`** — THE CORE METHOD:

       1. **Lock check**: Acquire `mu.Lock()`, check `inProgress[schoolID]`, set to true, release lock.
       2. **Load backup record**: Find `models.TenantBackup` by backupID and schoolID. Status must be `"completed"`. Return ErrBackupNotFound if not found.
       3. **Get school connection**: Load `models.SchoolConnection` from coreDB. Extract DSN (same pattern as BackupService). Use crypto.Service to decrypt password.
       4. **Download from S3**: Call `s3Storage.Get(backup.BackupPath)` to get the backup file. Write to a temp file using `os.CreateTemp("", "restore-*.dump")`.
       5. **Validate archive integrity**: Run `exec.Command("pg_restore", "--list", tempFile.Name())`. If exit code != 0, return ErrRestoreFailed with stderr output.
       6. **Restore to database**: Run:
          ```
          exec.Command("pg_restore", "--dbname="+dsn, "--clean", "--if-exists", "--no-owner", "--no-acl", tempFile.Name())
          ```
          - `--clean`: drop existing objects before recreating
          - `--if-exists`: avoid errors if objects don't exist yet
          - `--no-owner`: skip ownership restoration (handled by app)
          - `--no-acl`: skip ACL restoration
          - Timeout: 60 minutes (large databases)
       7. **Verify data integrity**: After restore completes successfully, run a pre-defined validation query set. One query per core table type:
          ```go
          validationQueries := []string{
              "SELECT COUNT(*) FROM students",
              "SELECT COUNT(*) FROM teachers",
              "SELECT COUNT(*) FROM classes",
          }
          ```
          Connect via `connManager.GetTenantDB(schoolID)` and execute each query. If any query fails, consider the restore as having verification issues but still mark it complete (the data is there, validation is informational).
          **Note:** The tenant tables may not exist immediately after restore if the migration system hasn't been run yet on the restored DB. The validation should be a best-effort check — if tables don't exist yet, log a warning but don't fail the restore.
       8. **Update backup record**: Set `RestorePoint` to current time.
       9. **Clean up**: Remove temp file. Set `inProgress[schoolID] = false`.
       10. **Log completion**: Log duration, size, backup path.

       **Helper methods:**
       - `getDSNForSchool(ctx context.Context, schoolID uint) (string, error)` — loads school, decrypts password, returns DSN (same logic as BackupService).
       - `runValidationQueries(ctx context.Context, schoolID uint) error` — connects via ConnectionManager, runs basic SELECT queries.

       ```go
       package handlers

       import (
           "context"
           "encoding/json"
           "fmt"
           "log"
           "time"

           "github.com/hibiken/asynq"
           "github.com/playbits/schoolcare-v2/internal/queue"
           "github.com/playbits/schoolcare-v2/internal/restore"
       )

       type RestoreTaskHandler struct {
           restoreService *restore.RestoreService
           logger         *log.Logger
       }

       func NewRestoreTaskHandler(svc *restore.RestoreService) *RestoreTaskHandler {
           return &RestoreTaskHandler{
               restoreService: svc,
               logger:         log.New(log.Writer(), "[RESTORE] ", log.LstdFlags),
           }
       }

       func (h *RestoreTaskHandler) HandleRestoreExecute(ctx context.Context, task *asynq.Task) error {
           var payload queue.RestoreExecutePayload
           if err := json.Unmarshal(task.Payload(), &payload); err != nil {
               return fmt.Errorf("restore handler unmarshal: %w", err)
           }

           start := time.Now()
           h.logger.Printf("Starting restore for school %d from backup %d", payload.SchoolID, payload.BackupID)

           if err := h.restoreService.RestoreTenantBackup(ctx, payload.SchoolID, payload.BackupID); err != nil {
               h.logger.Printf("Restore failed for school %d backup %d: %v", payload.SchoolID, payload.BackupID, err)
               return err
           }

           h.logger.Printf("Restore completed for school %d from backup %d (duration: %s)", payload.SchoolID, payload.BackupID, time.Since(start))
           return nil
       }
       ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| API → RestoreService | Authenticated admin user triggers restoration of a tenant database |
| RestoreService → pg_restore | Tenant database credentials used in subprocess |
| S3 → RestoreService | Backup files downloaded from S3 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-06 | Denial of Service | RestoreTenantBackup | mitigate | Prevent concurrent restores per school via inProgress map; set pg_restore timeout of 60 minutes |
| T-05-07 | Tampering | Backup download | mitigate | Validate archive integrity with `pg_restore --list` before restore |
| T-05-08 | Information Disclosure | DSN in logs | mitigate | Sanitize DSN before logging: log only `dbname={dbname} host={host}` without password |
| T-05-09 | Elevation of Privilege | API endpoints | mitigate | Enforce school_id scoping — user can only restore backups for their own school (middleware validates JWT school_id matches path schoolId) |
| T-05-10 | Information Disclosure | Backup listing | mitigate | ListBackups must only return backups for the authenticated user's school (schoolId validated against JWT) |
| T-05-11 | Data Corruption | Restore to wrong DB | mitigate | Double-check schoolID and backupID match; validate backup path belongs to the school (path includes schoolID) |
</threat_model>
