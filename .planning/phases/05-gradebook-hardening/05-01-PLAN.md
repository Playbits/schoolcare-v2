# 05-01: Grade Freeze Frontend UI

## Objective
When a session status is `completed`, show visual indicators in the frontend: a Frozen badge on session cards, a FrozenBanner on score pages, lock icons on disabled inputs. No backend changes — `canModifyScore()` + `disabled` prop already enforce read-only.

## Tasks

### Task 1: Frozen badge on session cards
1. Create `FrozenBadge` component: small badge with lock icon + "Frozen" label, muted color scheme
2. On `academics.tsx` session cards, add badge when `session.status === "completed"`
3. Position: top-right corner of session card, subtle appearance

### Task 2: FrozenBanner on score pages
1. Create `FrozenBanner` component: horizontal banner with lock icon + message "This session's grades are frozen. Contact admin to make changes."
2. On score/academics pages where `session.status === "completed"`, display banner at top of score area
3. Banner disappears on active sessions (status !== "completed")

### Task 3: Lock icons on disabled score inputs
1. ScoreGrid already has `disabled` prop — add lock icon (Lucide `lock` icon) alongside disabled inputs
2. Position: inline prefix on each disabled score input field
3. Tooltip: "Grades are frozen for this session"

### Task 4: Integration check
1. Verify badge + banner + lock icons all toggle correctly on session status change
2. Verify no regressions on active/upcoming session views
3. Verify admin/teacher sees same indicators (no role-based hiding — frozen is frozen for everyone)

## Files Modified
- `frontend/src/components/academics/score-grid.tsx` — add lock icons on disabled inputs
- `frontend/src/routes/_dashboard/academics.tsx` — add FrozenBadge on session cards, FrozenBanner on score pages
