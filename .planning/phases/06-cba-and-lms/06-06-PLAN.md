# 06-06: CBA → Assessment Integration

## Objective
Allow CBA assignments to be linked directly to `Assessment` objects in the academic module. When an assessment has a linked CBA assignment, completing the CBA exam automatically populates the assessment's scores — enabling schools to use CBA exams as CA (Continuous Assessment) or Exam components in the gradebook.

## Tasks

### Task 1: Add `cba_assignment_id` FK to `Assessment` model
1. Add nullable `cba_assignment_id` (uint → `CBAAssignment` model) to `Assessment` model
2. Add migration in school migration file
3. This is optional — existing assessments with no CBA link remain unchanged

### Task 2: Update Assessment DTOs
1. Add `cba_assignment_id` / `cba_assignment_uuid` to `CreateAssessmentRequest` DTO
2. Add `cba_assignment_id` + resolved CBA assignment name to `AssessmentResponse` DTO
3. Update `CreateAssessment` handler/service to accept and set the field

### Task 3: Score push from CBA to Assessment
1. In `pushScoreToGradebook` (from 06-01), after writing to `Score` table:
   - Check if the `GradeItem`'s parent `Assessment` has a `cba_assignment_id`
   - If so, also update/insert a `Score` record directly on the Assessment
2. This makes the CBA score appear as part of the assessment's grade breakdown
3. Uses same upsert pattern (ON CONFLICT student_id + assessment_id → update)

### Task 4: Frontend — Assessment editor CBA picker
1. On the assessment configuration page, add a "CBA Assignment" selector when creating/editing an assessment
2. Fetches `GET /cba/assignments` — shows a dropdown of available assignments
3. Teacher can link an assessment to a CBA exam in one click
