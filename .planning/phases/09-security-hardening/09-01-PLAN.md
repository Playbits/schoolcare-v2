    Rewrite `tenant.DSN()` at lines 18-25:
    ```go

            " port=" + itoa(c.DatabasePort) +
    ```

    Replace with `net/url` based construction:
    ```go
    import "net/url"
    
    func (c DBConfig) DSN() string {
        // Use net/url to properly encode the password and other values
        // This prevents DSN parsing issues when passwords contain special characters
        v := url.Values{}
        v.Set("sslmode", c.SSLMode)
        
        u := &url.URL{
            Scheme: "postgres",
            User:   url.UserPassword(c.DatabaseUsername, c.DatabasePassword),
            Host:   fmt.Sprintf("%s:%d", c.DatabaseHost, c.DatabasePort),
            Path:   c.DatabaseName,
            RawQuery: v.Encode(),
        }
        return u.String()
    }
    ```
    This produces a standard `postgres://user:password@host:port/dbname?sslmode=...` URI format.

    Wait — verify that GORM's `postgres.Open()` accepts this URI format. Check the pgx driver.
    GORM's postgres driver uses `pgx` which accepts both DSN (keyword=value) and URI (postgres://) formats.
    The `postgres.Open()` function in GORM accepts either format. Using URI format is safer.
    
    Actually, to be safer and not change the DSN format (which would affect the Connection Manager too), let me use the `net/url.QueryEscape` approach for just the password and other values individually:
    
    ```go
    import "net/url"
    
    func (c DBConfig) DSN() string {
        return fmt.Sprintf(
            "host=%s port=%d user=%s password=%s dbname=%s sslmode=%s",
            c.DatabaseHost,
            c.DatabasePort,
            url.QueryEscape(c.DatabaseUsername),
            url.QueryEscape(c.DatabasePassword),
            url.QueryEscape(c.DatabaseName),
            c.SSLMode,
        )
    }
    ```
    
    Hmm, but `pgx` keyword=value format doesn't use URL-encoded values. It uses libpq-style escaping.
    
    Best approach: use `net/url` to build a proper `postgres://` URI:
    ```go
    import (
        "fmt"
        "net/url"
        "strconv"
    )

    func (c DBConfig) DSN() string {
        u := &url.URL{
            Scheme: "postgres",
            User:   url.UserPassword(c.DatabaseUsername, c.DatabasePassword),
            Host:   net.JoinHostPort(c.DatabaseHost, strconv.Itoa(c.DatabasePort)),
            Path:   c.DatabaseName,
        }
        q := u.Query()
        q.Set("sslmode", c.SSLMode)
        u.RawQuery = q.Encode()
        return u.String()
    }
    ```
    
    This is the canonical approach. pgx's ParseURI and ParseDSN both handle this format.
    
    Keep the `itoa` helper since it may be used elsewhere, or remove if unused after.
