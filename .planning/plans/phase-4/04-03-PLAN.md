# 04-03: WAEC External Exam Integration

## Goal
Record WAEC/NECO external exam results (A1-F9 grades) with sitting numbers, credit count computation, and CSV import.

## Work Done
- **ExternalExamResult model** — `(admission_number, exam_type, sitting_number, subject)` composite unique, includes `grade`, `credit_flag`, `score`
- **CRUD endpoints**: `GET/POST /api/v2/external-exam/results`, `GET /api/v2/external-exam/results/best` (best per subject across sittings), `GET /api/v2/external-exam/results/credit-count` (A1-C6 count)
- **CSV import**: `POST /api/v2/external-exam/import/csv/preview` and `POST /api/v2/external-exam/import/csv/confirm` — preview-then-confirm with per-row validation
- **Grade validation**: A1, B2, B3, C4, C5, C6, D7, E8, F9 (WAEC), plus NECO equivalents
- **Credit flag**: auto-computed from grade (A1-C6 = true, D7-F9 = false)
- Group 22 migration creates `external_exam_results`
